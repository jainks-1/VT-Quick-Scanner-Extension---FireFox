# VT-Quick-Scanner-Extension---FireFox

Authors/Contributors:

* Kaden Jain
* Gemini

This browser extension allows a user to quickly check the safety rating of any URL/IP address/domain/file hash without exiting their current webpage. This extension utilizes VirusTotal, a free online service that scans suspicious files, URLs, IP addresses, and domains to detect malware and malicious content. 

It can be used on Mozilla FireFox.

The version of the browser it was created on is:

* Mozilla FireFox 150.0.2 (64-bit)
